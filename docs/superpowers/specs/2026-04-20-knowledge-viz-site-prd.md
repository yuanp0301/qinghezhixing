# 青禾知行 v1 产品 PRD

日期：2026-04-20
配套设计文档：`2026-04-20-knowledge-viz-site-design.md`
范围：v1 唯一菜单「互动动画管理」的全量页面与交互规范。

> 本 PRD 只描述产品层：页面结构、字段、交互、状态、文案、异常路径。技术实现见配套设计文档。

## 0. 通用规范

### 0.1 视觉与布局

整体追求**苹果式的清爽简洁**：克制、留白、去装饰、内容优先。

**布局**

- 三栏：**顶部导航（56px）** + **左侧菜单（220px）** + **主内容区**。
- 主内容区两侧留白 ≥ 32px，模块之间垂直间距 ≥ 32px。
- 响应式：优先保证 ≥1280px 体验；移动端仅保证「外部观看页」与「失效页」可用。

**色彩**

- 主色：清新青绿（建议 `#06B6A4` 一族），仅用于关键操作按钮、链接、选中态、品牌标识；用量克制，每屏不超过两处。
- 中性色为主：背景 `#FFFFFF` / 次级背景 `#F7F8FA` / 分隔线 `#EBEDF0` / 主文字 `#1D1D1F` / 次文字 `#6E6E73` / 占位 `#A1A1A6`。
- 状态色：成功 `#34C759` / 警告 `#FF9F0A` / 危险 `#FF3B30`，仅用于状态徽标和反馈，不大面积铺色。
- 不使用渐变、阴影叠加、彩色装饰；卡片仅用 1px 浅边或极浅阴影（`0 1px 2px rgba(0,0,0,.04)`）。

**字体与字号**

- 字体栈：`-apple-system, BlinkMacSystemFont, "PingFang SC", "Helvetica Neue", "Segoe UI", sans-serif`。
- 字号：H1 28 / H2 22 / H3 18 / 正文 14 / 次要 13 / 辅助 12。行高 1.5。
- 字重以 Regular / Medium 为主，避免 Bold 大量使用。

**圆角与控件**

- 圆角统一：卡片 12 / 按钮 8 / 输入框 8 / 标签 chip 6 / 头像圆形。
- 按钮：主按钮（实心主色 + 白字）、次按钮（白底 + 1px 边）、文本按钮（无边纯文字）。三种之外不要再造。
- 输入控件高度 36；按钮高度 36（主操作可 40）。
- 表格：无边框、行间分隔线，hover 行底色 `#F7F8FA`。
- 图标：线性单色（如 lucide / SF Symbols 风），1.5px 描边，统一大小 20。

**动画**

- 仅用于状态切换：淡入淡出 150ms、弹窗 200ms ease-out；不用弹跳/旋转/视差。

**风格反例（避免）**

- 多种主色并用、彩色背景大块铺设、立体阴影、渐变按钮、强对比霓虹色、emoji/插画装饰、过度的动效。

### 0.2 顶部导航

- 左侧：站点 Logo + 名称「青禾知行」，点击回公开库。
- 右侧：当前用户头像（首字母占位）+ 下拉菜单：`个人信息`（v1 只读展示用户名/角色）、`退出登录`。
- 未登录：访问需登录页面自动跳转登录页，并在登录成功后回跳原路径。

### 0.3 左侧菜单

- 一级菜单：`互动动画管理`（展开）
  - `公开库`
  - `我的上传`（仅 creator/admin 可见）
  - `管理后台`（仅 admin 可见）
    - `全部内容`
    - `用户管理`
    - `标签管理`
- 选中项以主色条 + 浅底色高亮。

### 0.4 全局状态与反馈

| 状态 | 样式 | 文案原则 |
|---|---|---|
| 加载中 | Skeleton / Spinner | "加载中..." |
| 空列表 | 空状态图 + 说明 + 主操作按钮 | 见各页面具体文案 |
| 成功操作 | 顶部 Toast（绿，3s 自动消失） | 动词 + 对象，如"上传成功" |
| 失败操作 | 顶部 Toast（红，5s） | 说明原因 + 可行动作 |
| 二次确认 | 居中弹窗 | 明确列出"将要发生什么" |
| 网络异常 | 内容区横幅 + 重试按钮 | "网络异常，请重试" |

### 0.5 全局时间格式

- 列表：`YYYY-MM-DD HH:mm`
- 相对时间（仅"创建于"等辅助信息）：`3 分钟前 / 2 小时前 / 昨天 10:00 / 2026-04-18`
- 倒计时（分发链接剩余时间）：`剩 23h 15m` / `剩 4 天` / `已过期`

## 1. 登录页 `/login`

### 1.1 布局

- 居中卡片（宽 400px），Logo + 标题「青禾知行」。
- 表单：
  - 用户名（必填，2–32 字符）
  - 密码（必填，8–64 字符，显示/隐藏切换）
  - 主按钮「登录」（全宽）
- 底部副文案：`如需账号，请联系管理员。`

### 1.2 交互

- 回车提交。
- 登录中按钮 loading + 禁用。
- 成功：写会话，跳转 `next` 参数指向的路径，默认「公开库」。

### 1.3 异常

| 场景 | 反馈 |
|---|---|
| 字段为空 | 字段下红色提示：`请输入用户名 / 请输入密码` |
| 账号/密码错误 | 表单上方红色横幅：`用户名或密码错误` |
| 账号被禁用 | 红色横幅：`账号已被禁用，请联系管理员` |
| 请求失败 | Toast：`登录失败，请稍后重试` |

## 2. 公开库 `/contents`

v1 默认首页。所有登录用户可见。

### 2.1 页面结构

- 顶部操作栏：
  - 搜索框（左，宽 320px）：按标题模糊匹配，回车或失焦触发。占位符 `搜索标题`。
  - 标签筛选（右）：多选下拉，显示已选标签数量。
  - 排序下拉（右）：`最新上传 / 最早上传`，默认最新。
  - （creator/admin 可见）右侧主按钮 `上传`（跳 `/contents/new`）。
- 内容列表：**网格卡片**，每行 4 个（1280px 下），卡片含：
  - 缩略区（灰底 + HTML 图标占位，v1 不做截图）
  - 标题（1 行省略）
  - 作者名 + 创建时间（相对时间）
  - 标签 chips（最多显示 3 个，超出显示 `+N`）
- 分页：底部居中，每页 24 条。

### 2.2 交互

- 点卡片 → 详情页 `/contents/{id}`。
- 筛选/搜索/排序变更：URL 同步 query 参数，刷新可复原。
- 清空筛选按钮（当有任何筛选条件时出现）。

### 2.3 空状态

- 无任何内容：`这里还没有互动动画` + 副文案 `联系制作者上传第一份内容吧`；creator/admin 角色额外显示 `上传` 主按钮。
- 搜索无结果：`没有匹配 "XXX" 的内容` + `清空筛选` 按钮。

## 3. 详情 / 观看页 `/contents/{id}`

站内登录用户访问。

### 3.1 页面结构

- 面包屑：`公开库 / {标题}`。
- 标题区：
  - 标题（H1）
  - 右侧操作（按权限显示）：
    - 作者本人或 admin：`编辑` / `生成分发链接` / `删除`
    - 其他登录用户：无操作
- 元信息栏（小字灰色）：
  - 作者名 · 上传时间 · 文件大小 · 原始文件名
  - 标签 chips（点击可回公开库并自动筛选）
- 观看区：
  - 容器比例 16:9，背景深灰
  - `<iframe sandbox="allow-scripts" src="/view/{id}">`
  - 右上角工具条：`全屏` / `在新窗口打开`
  - 首次进入显示"正在加载互动内容…"直到 iframe onload
- 简介区（若有）：Markdown 纯文本渲染（v1 不解析图片/链接）。

### 3.2 操作弹窗

- **编辑**：侧滑抽屉（440px），字段=标题/简介/标签，保存即时刷新。
- **删除**：二次确认弹窗 `删除后内容将不可访问，已生成的分发链接会一并失效。此操作不可撤销。` 按钮 `取消` / `确认删除`（红）。
- **生成分发链接**：见 §5。

### 3.3 异常

| 场景 | 反馈 |
|---|---|
| 内容不存在/已删除 | 整页空状态：`内容不存在或已被删除` + 返回公开库按钮 |
| iframe 加载失败 | 观看区替换为：`内容加载失败` + `重试` 按钮 |
| OSS 源文件丢失 | 与加载失败同，但后端日志标记 `source_missing` |

## 4. 上传页 `/contents/new`

仅 creator/admin 可访问；其他角色访问自动回公开库并 Toast 提示 `无权限`。

### 4.1 表单字段

| 字段 | 类型 | 必填 | 校验 | 说明 |
|---|---|---|---|---|
| 标题 | 单行文本 | 是 | 1–100 字 | |
| 标签 | 多选+可创建 | 否 | 每个 1–20 字，最多 10 个 | 下拉展示已有标签；输入不存在时回车新建（需 admin 未关闭自助建签时） |
| 简介 | 多行文本 | 否 | ≤ 500 字 | 纯文本 |
| HTML 文件 | 文件选择 / 拖拽 | 是 | `.html`，≤ 10MB，MIME=text/html | 双重校验 |

### 4.2 布局与交互

- 表单左，预览侧右（v1 预览区仅显示"上传后可观看"占位，不做真机预览）。
- 拖拽区：`将 HTML 文件拖到此处，或点击选择`；选中后显示文件名 + 大小 + `移除` 按钮。
- 提交按钮：`上传`（主）、`取消`（副）。
- 上传中：按钮 loading + 禁用，显示进度条百分比。
- 离开前若有未提交改动：浏览器 beforeunload 提示 `有未保存的更改，确定离开？`。

### 4.3 成功路径

- 上传成功 → Toast `上传成功` → 跳转详情页 `/contents/{id}`。

### 4.4 异常

| 场景 | 反馈 |
|---|---|
| 文件类型不符 | 字段下红色：`仅支持 .html 文件` |
| 文件过大 | 字段下红色：`文件大小不能超过 10MB`（当前 X MB） |
| 标签超限 | 字段下红色：`最多 10 个标签` |
| 标题重复 | 允许同名，不拦截（仅靠 file_id/sha256 区分） |
| 网络中断 | Toast 红：`上传失败，请检查网络后重试`；表单保留已填内容 |
| 服务端 5xx | Toast 红：`服务异常，请稍后重试` |

## 5. 分发链接管理

v1 不做独立页面，入口与展示都挂在**详情页**上。

### 5.1 「生成分发链接」弹窗

- 触发：详情页右上操作 `生成分发链接`（仅作者本人 / admin 可见）。
- 字段：

| 字段 | 类型 | 默认 | 说明 |
|---|---|---|---|
| 有效期 | 单选 | 24 小时 | 选项：`1 小时 / 24 小时 / 7 天 / 自定义` |
| 自定义时长 | 数字 + 单位 | — | 仅"自定义"选中时出现；范围 5 分钟 – 30 天 |
| 允许下载原文件 | 开关 | 关 | 说明文案：`开启后，访客除了在线观看还可下载 HTML 源文件` |

- 按钮：`取消` / `生成`。

### 5.2 生成结果

- 生成成功后弹窗替换为"结果态"：
  - 大号展示 URL（只读）
  - `复制链接` 按钮（成功 Toast `已复制`）
  - 副文案：`链接将在 {expires_at} 过期` + 复制含时间的整段文案按钮 `复制链接和有效期`
  - 底部：`完成` 按钮关闭弹窗
- 同时"分发链接列表"实时新增一行。

### 5.3 详情页内嵌的「分发链接列表」

- 仅作者本人 / admin 可见，位于观看区下方折叠区块 `分发链接（N）`。默认折叠，标题右侧显示 `有效 N / 总 M`。
- 列表字段：

| 列 | 说明 |
|---|---|
| 链接 | 短展示 `/s/{token前 8 位}…{后 4 位}`，hover 显示完整；点击复制 |
| 状态 | `有效 / 已过期 / 已撤销`，彩色标签 |
| 倒计时 | 仅"有效"状态显示剩余时长 |
| 允许下载 | `是 / 否` |
| 生成时间 | 绝对时间 |
| 访问次数 | 数字，点击跳"访问记录"抽屉 |
| 操作 | `复制` / `撤销`（仅有效时可点） |

- 撤销二次确认：`撤销后此链接立即失效，已通过它打开页面的访客刷新后将看到失效提示。确定撤销？`

### 5.4 「访问记录」抽屉

- 右侧抽屉（520px），标题 `分发链接访问记录`。
- 列表字段：`访问时间 / 访客 IP（脱敏后两段，如 192.168.*.*） / User-Agent（缩略） / 结果`
- 结果列：`成功 / 已过期 / 已撤销 / 不存在`
- 分页，每页 20 条。
- 空状态：`暂无访问记录`。

## 7. 外部观看页 `/s/{token}`

无需登录。这是 v1 对站外访客唯一的入口。

### 7.1 有效态

- 极简布局：顶部一条窄横幅 + 全屏观看区。
- 横幅：左侧 Logo「青禾知行」（无链接）；右侧 `剩余有效期：23h 15m`（每 60s 刷新一次）。
- 观看区：同详情页 iframe sandbox，但隐藏全屏/新窗口按钮以外的所有操作。
- 如 `allow_download=true`：观看区右上角增加 `下载原文件` 按钮，点击调 `/d/{token}`。
- 无作者信息、无标签、无评论——保持"阅后即焚"的克制感。

### 7.2 失效态（统一页）

过期 / 撤销 / 不存在 / 格式错误，一律返回同一页面：

- 居中图标 + 标题：`链接已失效`
- 副文案：`此链接可能已过期、被撤销，或从未存在。请联系发送方获取新链接。`
- 底部小字：`由 青禾知行 提供`
- **不显示**具体失效原因，防止侧信道。

### 7.3 限流触发

若触发 IP 限流：

- 返回 429 页：`访问过于频繁，请稍后再试`
- 不暴露限流阈值细节。

## 8. 管理后台

仅 admin 可见。三个子页共用顶部 `返回站点` 链接。

### 8.1 全部内容 `/admin/contents`

- 与"我的上传"同表格结构，但无"只看自己"过滤，多出两列：
  - 上传者（用户名 + 链接跳用户管理筛选该用户）
  - 状态（`正常 / 已删除`，支持筛选）
- 操作列额外 `下架`（软删）/ `恢复`（对已删除项）。
- 二次确认文案同详情页删除。

### 8.2 用户管理 `/admin/users`

- 顶部：搜索框（用户名）、角色筛选、状态筛选、右侧 `新建用户` 主按钮。
- 表格：`用户名 / 角色 / 状态 / 创建时间 / 最近登录 / 操作`
- 操作：`编辑` / `重置密码` / `启用/禁用`。
- **新建用户**抽屉：

| 字段 | 必填 | 校验 |
|---|---|---|
| 用户名 | 是 | 2–32 字，英数下划线，唯一 |
| 初始密码 | 是 | 8–64 字，要求至少包含字母与数字 |
| 角色 | 是 | `viewer / creator / admin` 单选，默认 viewer |
| 备注 | 否 | ≤ 200 字 |

- **重置密码**：弹窗生成 12 位随机密码，一次性展示 + `复制` + 说明 `请线下转交用户，关闭后无法再次查看`。
- **禁用**：二次确认 `禁用后该用户无法登录，已有会话将在下次请求时失效。`

### 8.3 标签管理 `/admin/tags`

- 表格：`标签名 / 关联内容数 / 创建时间 / 操作`
- 操作：`重命名` / `合并到…`（下拉选另一个标签） / `删除`（仅关联为 0 时可删）
- 顶部：搜索 + `新建标签` 按钮。
- 合并语义：源标签被删除，所有引用改指向目标标签。二次确认：`将 "A" 合并到 "B" 后，A 会被删除，N 条内容的标签会被改为 B。`
